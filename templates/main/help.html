Functions for the SA

New Entry

The Record number, SA, and Date are all auto populated with the current timestamp. In the field are two text areas to enter additional information. The Entry field is for any notes, comment, dates, and relavant data.

The Comment/Issues area is for anythin that may casue an imapct into what the event is about.

Once complete click the button at the bottom of the screen.

List Entry

List all the entries in the database for the current SA.

Delete Entry

Choose the entry number to delete.

Edit Entry

Choose the entry number to Edit.

Edit Entry

Choose the entry number to Edit.

Clone Entry

Choose the entry number to Clone. Cloning an entry copies the record to a new date and record number. Use this if you were working on an entry in the past and you have returned to the same activity.

Functions for the Manager

List All

Lists all the records in the database.

List By SA

Lists all the records for a single SA.

Download Excel

Allows the manager to download all the records for a specific date forward.
